Y-axis Limits

The Y-axis Limits tab sheet provides the options to change the minimum and maximum limits for the Y-axis or the Y-axis scale. The tab sheet is visible for only axes checked in the Plot column. One reason for changing the limits is to produce a series of diagrams for the same span of time. Below are some examples.

In this example the program automatically determines all the limits. The minimum and maximum axis limits are the minimum and maximum levels or Y-axis values. The program calculates rounded values for the minimum and maximum scale limits. The Axis Limits are enabled for only the primary axis (the first one in the list).

 

 

In this example the maximum limit for the scale is set to 18,000. Note that only the scale extends to 18,000.

 

 

In this example the maximum limits for both the axis and scale are set to 18,000. Note that both the axis and the scale extend to 18,000.
