Numbers Graph

Click in the Graph Style Dialog Box to select the Numbers graph style and to activate the options for this style. In a Numbers graph, the actual value is printed for each sample point. Numbers will be offset to avoid overlapping. The most common uses for Numbers Graphs are for sums, as in the examples here, or for small values that are not visible in a silhouette, histogram, or line graph, as in the example here. If the Show Zero Values check box is checked, zero values will be printed, as in the example here.
