Variables Dialog Box

The Variables Dialog Box is accessed by clicking Variables under the Options menu or by clicking on the toolbar. In this dialog box, variable names can be changed; the order of variables can be changed; and variables to be overlaid, stacked, or skipped can be designated. The variable order can also be changed interactively.

 

 

 

 

The Variables Dialog Box provides the following options:

 

Change the order of variables on the diagram

Overlay variables

Stack variables

Skip variables

Edit variable names

Format variable names
