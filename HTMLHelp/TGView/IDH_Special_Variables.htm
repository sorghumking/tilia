Special Variables

Tilia recognizes some special variables that have reserved VarCodes. The special status of these variables is denoted by their VarCodes; their VarNames for can be anything. These variables are not listed in the Variables Dialog Box.

  1. Chronology variables, which have the VarCode ChronX, where X is a number (e.g. Chron1), contain the age for each sample. Multiple chron variables can exist for different chronologies, e.g. Chron1 in radiocarbon years and Chron2 in calibrated years. These variables are listed in the Y-axes Dialog Box, where Y-axes and scales are designated.

  2. Depth variables, which have the VarCode DepthX, where X is a number (e.g. Depth1), contain the depth for each sample. Multiple depth variables can exist for different depth reference points, e.g.Depth1 could be depth from water surface and Depth2 could be depth from sediment surface. Normally, the sample numbers are depths, and a depth variable is not necessary. However, a depth variable is necessary if multiple possibilities exist for depths. These variables are listed in the Y-axes Dialog Box, where Y-axes and scales are designated.

  3. Thickness variables have the Varcode Thick or ThickX, where X is a number (e.g. Thick1). The number is not needed for a thickness variable and is ignored if provided. Only one thickness variable is possible. The thickness variable contains the thickness of each sample in the units of depth measurement (e.g. cm). Thickness variables can be used for bar thicknesses in Histogram Style graphs.
