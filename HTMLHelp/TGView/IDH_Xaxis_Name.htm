X-Axis Name

 

 

Set options for the X-axis name on the Name tab sheet in the X-axes Dialog Box. The X-axis name is the name of the variable that appears above the graph. The name of the variable can be changed in the Variables Dialog Box.

 

Options:

  • The angle can be between 0° and 90°.

  • Click to change the font.

  • The offset is from the top of the Y-axis. Offsets are in virtual units.

  • Click the Center check box to center the name on the graph.
