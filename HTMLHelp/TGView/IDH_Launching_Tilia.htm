Launching Tilia

Tilia is a DOS spreadsheet program for entering and manipulating pollen data or other kinds of similar data. The data are saved to a binary formatted *.til file, which TGView can read. In the future, the spreadsheet functionality of Tilia will be added to TGView. For the moment, however, Tilia is required for producing the initial *.til data file for TGView. Many users of TGView will already have installed Tilia on their computers, and Tilia may be run independently from the DOS command prompt or from a desktop icon. Alternatively, Tilia may be launched from TGView, by clicking . The following dialog box will appear:

 

 

Select the working directory from the tree view of the directory structure. This control behaves similarly to Windows Explorer. The selected directory is shown in the edit line above. The directory may be typed directly in the edit line. If the directory does not exist, a message box will appear asking if the directory should be created.

 

When the program is run the first time, the default working directory will be the directory in which TGView is installed. It is highly recommended that data files not be saved in this directory, but rather in a special data directory, perhaps under My Documents.
