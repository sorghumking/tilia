Stack Graphs

 

 

 

 

Stack variables with one of the following procedures:

  1. Select the graphs to be stacked and click . This procedure is illustrated above. The Stack button will not be enabled until two or more consecutive variables are selected.

  2. Click the Base check box for the first variable and the Stack check box for additional variables below the base variable. No variables may intervene between the base and stacked variables.
