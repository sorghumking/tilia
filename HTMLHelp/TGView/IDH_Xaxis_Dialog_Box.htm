X-axes Dialog Box

The X-axes Dialog Box is accessed by clicking X-axes under the Options menu or by clicking on the toolbar.

 

 

 

Before changing any options, you must first select one or more graphs by clicking . The box to the right of the button indicates if variables have been selected. If one variable has been selected, the box will contain its name. Otherwise the number of selected variables will be listed.

 

Select the tab sheet for one of the following options:

 

Axis

Tic Marks

Name

Truncate

 

 

Click or to accept the changes made. Click to go to the Graph Style Dialog Box for the same selected variables.
