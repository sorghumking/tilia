Select Y-axes

 

 

This list box lists all the Y-axes in the spreadsheet. The default Y-axis is defined by the sample numbers in the spreadsheet. Other axes are defined by variables having the VarCodes ChronX or DepthX, where X = an integer number, e.g. Chron1, Chron2, Depth1.

 

The first axis listed is the primary axis. All scaling is relative to the primary axis. Only axes checked in the Plot column will be plotted on the diagram. The Code cannot be edited, but you can edit the Name, which will be the name of the scale on the diagram.

To change the order of axes on the diagram, click or , and the highlighted axis will move.
