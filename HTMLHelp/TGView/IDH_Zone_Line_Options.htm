Zone Line Options

 

Options for Zone Lines include:

  • Line style (solid, dotted, short dashed, long dashed, user defined).

  • Line width in actual points.

  • Color.

 

Click on the line type desired. For User Defined lines, the Dash Width and Space Width are measured in virtual points.

 

User defined lines will be scaled exactly the same on the display and the printer relative to the rest of the diagram. The other line styles are measured in pixels and are, therefore, device dependent; and will appear differently on different devices.

 

Check Apply to all line zones to apply the selected options to all line zones.
