Print Truncated Values on the X-axis

 

The truncated values can be printed by checking the Print Truncated Values check box.

 

The printed values can be scaled the same as the tic mark labels by checking the Scale Values as Tic Marks check box. The scaling matters only if Divide by is not equal to one on the Tic Marks tab sheet.

 

Click to change the font. Font size is measured in virtual points.
