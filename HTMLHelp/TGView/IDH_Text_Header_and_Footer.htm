Header & Footer

 

 

The Header can have up to four lines. If Automatic is checked for the X-Coordinate and Y-Coordinate, the header is placed in the upper left-hand corner of the diagram.

 

The Footer has only one line. If Automatic is checked for X-Coordinate and Y-Coordinate, the footer is placed in the lower right-hand corner of the diagram.

If Automatic is checked for any coordinate, the coordinate is calculated after is clicked. The calculated coordinate position will be displayed the next time the Text Dialog Box is shown. If changes are made to the diagram, Automatic will have to be rechecked to replace the header or footer to its default position.

 

Text can be formatted with the font formatting buttons.

 

Click to change the font.

 

 

Tip

  • Initially check the Automatic check box to position the Header or Footer and click to accept. Then return to the Text Dialog Box to reposition.
