Greek Letters Dialog Box

 

 

Click the letter to be inserted in the text string. The name of the letter with the <g></g> tags will appear in the text box. The letter will be inserted at the cursor position in the edit line of the calling dialog box. The case of the Greek character is indicated by the case of the name, for example:

 

<g>SIGMA</g>

S

<g>sigma</g>

s
