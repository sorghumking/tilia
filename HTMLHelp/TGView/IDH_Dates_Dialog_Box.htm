Dates Dialog Box

The Dates Dialog Box is accessed by clicking Dates under the Options menu or by clicking on the toolbar.

 

 

Enter the Top Depth and Bottom Depth for each date. To enter a plus-or-minus (±) sign, click , which is enabled when the Date column is active.

 

 

Insert or delete dates with the following buttons:

 Insert date.

 Delete date.

 

 

Tips

  • To enter the first date in an empty worksheet, press .

  • To sort the dates by depth, click the Top Depth column header. A second click reverses the sort order (ascending order is changed to descending and vice versa). The small arrow displayed in the column header indicates the current sort order. The up-arrow indicates the ascending sort order, the down-arrow the descending sort order.

  • Before specifying User Defined coordinates, place the label approximately where desired by selecting Automatic Side or Automatic Top and clicking to accept. Then return to the Dates Dialog Box and reposition the label where desired.
