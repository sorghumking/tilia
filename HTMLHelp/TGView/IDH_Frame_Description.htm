Diagram Frame

The Diagram Frame is rectangle drawn around the diagram. The Frame Margin is the distance in virtual units between the frame the maximum and minimum limits of the diagram. The default value for the frame margin is 0.1. The Page Margin is the distance in virtual units between the edges of the diagram, including the frame, and the page edges. The default values for the page margins are 0.0. The Frame and Page Margins are designated in the Frame Dialog Box.

 

 

 

 

Notes

  • Because the page margins are in virtual units, the distance in actual units will depend on the size of the diagram.

  • Because the diagram is plotted in the lower left corner of the page, the bottom and left page margins will be as designated; however, the top and right page margins are minimum values.
