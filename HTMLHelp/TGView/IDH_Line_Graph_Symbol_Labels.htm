Line Graph Symbol Labels

Labels for line graph symbols are shown if the Symbol Labels check box is checked on the Line Graph tab sheet in the Graph Style Dialog Box. Options exist for label position and for plotting the graph name in addition to the symbol name. See the examples below for the various options.

Notes

  • The X and Y coordinates are relative to the graph origin, not the diagram origin.

  • The first time Symbol Labels is checked and Automatic Top or Automatic Bottom is selected, the User Defined Coordinates will all be 0. In subsequent activations of the Graph Style Dialog Box, the automatically assigned coordinates will be shown but will not be enabled unless User Defined is selected.

 

Tip

  • Before setting User Defined coordinates, first select Automatic Top or Automatic Bottom and apply. This action will set the X and Y coordinates, which you can then modify.
