Cluster Analysis Dialog Box

The Cluster Analysis Dialog Box is accessed by clicking Cluster Analysis under the Options menu or by clicking on the toolbar.

 

 

 

If a diagram window is not open, only the button will be enabled. If a diagram window is open, the button will also be enabled. If a .dgx file has been loaded, the rest of the dialog box will be enabled.

 

 

The Cluster Analysis Dialog Box has the following options:

 

Run CONISS

Open a .dgx File

Plot dendrogram

Dendrogram scale

Axis label

Title

Tic mark step size

Tic mark labels

Tic mark lengths
