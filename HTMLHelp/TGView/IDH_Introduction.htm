Introduction

TGView is a Windows program that reads Tilia *.til files and Tilia•graph *.tgf files. The program will read *.til files version 2.x and *.tgf files versions 1.10 – 1.26. Version 2.0 of TGView completely replaces the DOS program Tilia•graph. However, Tilia is still required to produce the initial data file. TGView will save your diagram to a new *.tgx format file, which it will also read. In the future, the spreadsheet functionality of Tilia will be added to TGView.
