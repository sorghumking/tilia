Zone Screen Options

 

A Zone Screen is actually a long rectangle with a fill. The fill can be Solid, Pattern, or Dashed. For a Dashed zone, the dash width and space width are specified in virtual points.

 

The Zone Width is specified in either in virtual or depth units. Depth units are the units of the Y-axis, e.g. cm. Virtual units are the arbitrary units in which the diagram is constructed.

 

If the Draw Borders check box is checked, borders are drawn around the zone bar. Borders also have options for line width (in actual points) and color.

 

Click the Zone Color button to change the zone color.

 

Check the check box for Apply to all screen zones to apply the selected options to all screen zones.

 

 

 

This example shows a Dashed zone 0.1 Virtual units thick with no Borders (Dash Width = 20, Space Width = 20) and a Pattern zone 40 Depth units (cm) thick with Borders (the options in the dialog box above). The dashed zone has To Back order, and the pattern zone has To Front order.

 

 

Tip

  • To blank out a stratigraphic section of the diagram, use a screen zone with a solid or pattern fill and set order to To Front.
