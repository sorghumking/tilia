Random Text

Text can be placed anywhere on the diagram in the Random Text tab sheet in the Text Dialog Box. This tab sheet will also display interactive text, which can be edited.

 

 

 

Enter the Text, X-Coordinate, Y-Coordinate, and Angle. Under Font, click to change the font. To see an example of the text entered above click here.

 

Text can be formatted with the font formatting buttons.

 

Insert or delete levels with the following speed buttons:

 Insert new row.

 Delete row.
