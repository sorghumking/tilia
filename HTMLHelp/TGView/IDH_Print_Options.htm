Print Options Dialog Box

 

The Print Options dialog box provides the following options:

 

Fit to page The diagram is printed as large as possible for the selected page size and orientation, which are set with Properties in the Print dialog box.

 

Page height The diagram is printed to the maximum page height and will be printed in sections on two or more pages if necessary. A percent overlap can also be set.

 

Diagram height Set the total diagram height, including text, in either inches or cm. The diagram will be printed in two or more sections if necessary.

 

Y-axis height Set the height of the Y-axis in either inches or cm. This option is useful for providing the same vertical scaling for different diagrams. For example, if the Y-axis of Diagram A is 12,000 yrs and the Y-axis of Diagram B is 10,000 years, setting the Y-axis height for A to 6 inches and for B to 5 inches will provide the same vertical scaling for both diagrams.
