X-axis Minimum Limit

The minimum value of the X-axis is set in the Minimum Axis Limit edit box on the Axis tab sheet in the X-axes Dialog Box. If the Automatic check box is checked, the program will automatically determine the minimum axis limit. The automatic value is zero if the minimum data value is > 0 or = 0. If the minimum data value is < 0, the automatic value is the first major tic mark less than the minimum data value.
