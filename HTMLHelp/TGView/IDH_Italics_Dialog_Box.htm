Italics Dialog Box

 

To italicize a group of graph names at once:

  1. Highlight a block of variables in the Variables Dialog Box.

  2. Click .

 

The Italics Dialog Box offers the following choices:

 

 Italicizes all variable names in the block

 Removes all italics in the block

Automatically italicizes genus and species names.

 

Auto Italics uses the following guidelines:

  1. Subfamily, family, and order names with the suffixes oideae, aceae, and ales are not italicized.

  2. Names with a SUM VarCode are not italicized.

  3. Greek letters are not italicized.

  4. –type is not italicized.

  5. undiff. is not italicized.

  6. subg. and subgen. are not italicized.

  7. cf. is not italicized.
