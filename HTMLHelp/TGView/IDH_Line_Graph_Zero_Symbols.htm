Show Symbols for Zero Values on Line Graphs

Symbols are shown for zero values if the Show Symbols for Zero Values check box is checked on the Line Graph tab sheet in the Graph Style Dialog Box. If this check box is unchecked, zero values will not be marked. if this box is checked, very small values may not be distinguishable from zero.
