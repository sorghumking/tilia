Graph Style Dialog Box

 

The Graph Style Dialog Box is accessed by clicking Graph Style under the Options menu or by clicking on the toolbar. This dialog box provides options for changing the graph style.

 

 

 

 

Before changing any options, one or more graphs must be selected by clicking . The box to the right of the button indicates if variables have been selected. If one variable has been selected, the box will contain its name. Otherwise the number of selected variables will be listed.

 

Click one of the graph style buttons to select the graph style and to activate the options for that style:

 

Silhouette

Histogram

Line Graph

Presence

Numbers

 

 

Click to apply the options to the selected graphs. Click to go to the X-axes Dialog Box for the same selected variables.

 

 

Note

  • If more than one variable is selected, options that are different will not be changed unless you explicitly change them.
