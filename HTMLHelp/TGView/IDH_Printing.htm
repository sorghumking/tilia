Printing

To print a diagram:

  • Click Print on the File menu, or

  • Click on the Standard tool bar.

 

Normal print options such as page size and page orientation can be set by clicking the Properties button in the standard Windows Print dialog box. For additional options, click Print Options on the File menu.

 

 

 

 

The Print Options dialog box provides the following options:

 

Fit to page: The entire diagram is scaled to be as large as possible for the page size and page orientation.

 

Page height: The diagram is scaled to the page height and will be printed over multiple pages if necessary. % Overlap indicates the percent that will be overlapped on each page if the diagram is longer than one page.

 

Diagram height: The diagram height is designated in inches or cm. This height includes all graphic elements. The diagram will be printed over multiple pages if necessary.

 

Y-axis height: The Y-axis height is designated in inches or cm. This height is the distance between the Minimum and Maximum Axis Limits, which are designated in the Y-axes Dialog Box. The diagram will be printed over multiple pages if necessary. This option is particularly useful for printing different diagrams with the same vertical scaling. For example, if the vertical span of one diagram is 6,000 years and another is 10,000 years, printing the first diagram with a Y-axis height of 6.0 and the second diagram with a Y-axis height of 10.0 will produce diagrams with the same vertical scaling of 1 cm/1000 yr.

 

Units: Units of measurement for Diagram Height and Y-axis Height.

 

Orientation: Orientation of the diagram on the page. If no orientation is selected, the default orientation of the printer is used. If an orientation is designated, it will override the default printer orientation.
