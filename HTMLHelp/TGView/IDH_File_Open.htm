Opening a File

To open a file, either:

  • Click Open on the File menu, or

  • Click on the Standard tool bar.

 

The program will open three kinds of files:

  1. .tgf files versions 1.10 – 1.26, which are written by the DOS program Tilia•graph.

  2. .tgx files, which are written by TGView. These files have an XML format.

  3. .til files, which are spreadsheet files written by Tilia.

.tgf and .tgx files contain the instructions for producing a diagram. These are not graphics files or metafiles that can be read by other graphics programs. .til files are DOS Tilia spreadsheet files. When TGView reads a .til file, it produces a default diagram, which can be edited and saved to a .tgx file.
