Silhouette Graph Exaggeration

 

Silhouette graphs may be exaggerated in order to make small values more visible. The exaggeration factor may be set to any positive integer between 0 and 100 on the Silhouette tab sheet in the Graph Style Dialog Box. Set Factor to zero for no exaggerated curve. The example below shows the various Fill Styles for exaggerated curves.

 

 

Picea - Solid

Fraxinus - Hollow

Betula - Pattern

Ulmus - Depth Bars

Acer - Solid

Ostrya/Carpinus - Grid Lines (Minor Tic Marks)

Quercus - Grid Lines (Major Tic Marks)

 

 

Notes

  • The Grid Line Spacing radio group is enabled only if the Grid Lines fill style is selected.
