Y-axis Tic Marks

 

 

The Y-axis Tic Marks tab sheet has options for:

  • The step size for minor and major tic marks (major tic marks are labeled). If Automatic is checked, the program calculates rounded values for major tic marks, and minor tic marks = major tic marks.

  • The lengths of the minor and major tic marks in virtual units.

  • The distance (in virtual units) from the tic mark to the tic mark label.

  • A factor by which to divide the tic mark label. In the example above the tic mark labels are divided by 1000.

  • The number of decimal points in the tic mark labels.

  • The Font of the tic mark labels.
