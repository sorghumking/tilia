X-axis Length

 

 

The following options exist for setting the Axis Length on the Axis tab sheet in the X-axes Dialog Box.

  • Automatic: The X-axis is scaled automatically. If the maximum X value is less than or equal to 100, the scale factor will be 0.04. If the maximum X value is greater than 100, the graph width is set to 3.0 virtual units.

  • Fixed Width: The graph width is set to a fixed width in virtual units. The tic marks will be scaled to the fixed width.

  • Scale Factor: The scale factor is set in virtual units/natural units. For the default scale factor of 0.04, a graph with a maximum X value of 100 would have a width of 4 virtual units (Scale Factor = 4/100). With a scale factor of 0.02, the graph would be half the width.
