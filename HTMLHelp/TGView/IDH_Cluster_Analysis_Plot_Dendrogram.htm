Plot Dendrogram

Check Plot Dendrogram to plot a dendrogram on a diagram. This check box is enabled only if a .dgx file has been opened. It will be checked when a new .dgx file is opened. Unchecking the box will disable drawing of the dendrogram but will not unload the dendrogram data. It can be rechecked to draw the dendrogram without reopening a .dgx file.
