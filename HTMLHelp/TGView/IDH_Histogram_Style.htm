Histogram Style

 

The Style of histogram graphs is designated on the Histogram tab sheet in the Graph Style Dialog Box.

The Style for histogram graphs can be Lines or Bars.

  • If the Style is Lines, the graphic element is a line, and the Line Width can be designated. The line width should be no thicker than the width of the axis line.

  • If the Style is Bars, the graphic element is a rectangle, and the Bar Thickness and Fill Style can be designated. Check the Use Thickness Variable to use a thickness variable to determine bar thickness. Otherwise, bar thickness is in virtual units, and the minimum bar thickness is 0.02. is activated only if the Pattern style is selected.
