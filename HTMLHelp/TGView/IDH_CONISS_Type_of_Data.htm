The Type of Data can be Counts, Percentages, or Proportions. For pollen counts,  Convert to proportions is normally checked. However, the spreadsheet could include non-frequency data that should not be converted to proportions. Percentages are automatically converted to proportions. For pollen data, the best option is usually to use count data and have CONISS convert the data to proportions, which will insure that the sums add to 1.0 (100%). See the warning about the kinds of data that can be included in spreadsheet (.til file).
