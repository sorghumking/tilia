Lithology Options

 

 

The Options tab sheet in the Lithology Dialog Box contains the controls for designing the lithology key and various options for the column.

 

To provide a key to the patterns for lithologic components, click . All the components specified on the Design tab sheet will be imported. Each component checked in the Plot column will have rectangular box drawn showing the pattern. The labels for the key boxes are entered in the Label column. Lithologic components cannot be entered directly, and the Component column cannot be edited. By default all of the lithologic components will have a key, and the checkbox in the Plot column will be checked. However, it may not be necessary to key different proportions of the same component, and the Plot checkbox can be unchecked. In the example above, Limus detrituosus (gyttja) is represented in three different proportions, but only one is used in the key. The X and Y coordinates are automatically assigned, but these can be changed. The label is initially blank.
