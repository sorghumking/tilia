Presence Graph

Click in the Graph Style Dialog Box to select the Presence graph style and to activate the options for this style. In a Presence Graph, a symbol (filled circle) is drawn at each level at which the variable has a value greater than zero. A presence graph will show small values that would not be visible in a silhouette, histogram, or line graph. Another use is to show levels where macrofossils occur.
