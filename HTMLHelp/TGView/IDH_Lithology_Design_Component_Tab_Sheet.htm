Lithology Component Tab Sheet

Click a tab to select bitmap patterns or one of the Troels-Smith vector patterns. The Troels-Smith patterns are arranged four to a tab sheet. Each tab sheet is a lithologic component, and each of the four patterns represents the proportion of the sediment comprised of that component. This example is of Limus detrituosus (gyttja). The code is Ld and the number following the code indicates the approximate proportion: 1 ~ 25%, 2 ~ 50%, 3 ~ 75%, 4 ~ 100%. Click a pattern to select it. The selected pattern will then appear depressed.

 

The selected pattern is displayed in the example, and it can be shifted with the shift buttons, or its color can be changed. To return to a pattern already selected, click the appropriate tab, and the selected pattern for that component will appear depressed. This pattern can then be shifted, or its color changed.

 

Some components can also have a humicity classification. In the Troels-Smith scheme, humicity can have values from 0 to 4. The default humicity is 0. Greater humicity is indicated by thicker lines in the pattern. To designate a humicity >0, click one of the humicity buttons: . Darker gray indicates greater humicity. The clicked button will appear depressed and will show the number on the humicity scale, eg. . Humicity is not indicated in the example panel due to limitations of display resolution. Thicker lines may not display well on a monitor, but will have the correct appearance on a high-resolution printer (300 dpi or greater).

 

Bitmap patterns are all shown on the Bitmaps tab sheet. Click a pattern to select it.
