Y-axis Label

 

 

 

The Y-axis Label tab sheet has options for:

  • The label position

  • Coordinates and angle of label if the Position is User Defined

  • Font of the label.

If the position is Automatic Side, the angle is set to 90°, and the label is centered to the left of the axis. If the position is Automatic Top, the label is plotted above the axis, and the text angle can be set. If the position is User Defined, the user can set the X and Y coordinates and text angle.
