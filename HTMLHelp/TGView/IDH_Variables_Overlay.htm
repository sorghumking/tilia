Overlay Variables

 

 

 

 

Overlay variables with one of the following procedures:

  1. Select the graphs to be overlaid and click . This procedure is illustrated above. The Overlay button will not be enabled until two or more consecutive variables are selected.

  2. Click the Base check box for the first variable and the Overlay check box for additional variables below the base variable. No variables may intervene between the base and overlaid variables.

 

Variables Dialog Box

 

The graph for an overlaid variable is plotted on top of the graph for the preveious variable. In the example to the left, the graph for Pinus strobus, which is in histogram style, overlays the graph for total Pinus, which is in silhouette style.
