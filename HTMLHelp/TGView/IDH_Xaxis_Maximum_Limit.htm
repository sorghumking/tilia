X-axis Maximum Limit

The maximum value of the X-axis is set in the Maximum Axis Limit edit box on the Axis tab sheet in the X-axes Dialog Box. If the Automatic check box is checked, the program will automatically determine the maximum axis limit, which will be the first major tic mark greater than the maximum data value.

 

Tip

  • If the Maximum Axis Limit is less than the maximum data value, the graph will overlap the next graph to the right. This procedure can save space on the diagram.
