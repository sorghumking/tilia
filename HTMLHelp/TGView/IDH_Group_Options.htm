Group Options

 

 

The Group Options dialog box has the following options:

 

Text position: The group name may be In Line or Above Line. Above Line is a good option if the group name is longer than the line. See the examples below.

 

 

Lines: Options for line width and color.

 

Font: Click to change the font of the group labels in the Font Dialog Box.

 

Click the Apply to all groups check box to apply the options to all the groups.
