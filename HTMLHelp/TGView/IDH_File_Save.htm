Saving a File

To save a file, either:

  • Click Save or Save As on the File menu, or

  • Click on the Standard tool bar.

 

The program saves to a .tgx file, which has an XML format. Save As shows the Save As dialog box, in which a file name can be entered. Save saves to the existing file name without asking for a name. If the file has not been previously saved as a .tgx file, Save defaults to Save As. The button behaves as Save.
