Y-axes

 

More than one scale can be plotted. The scaling of the Y-axis is based on the first axis in the list box. All other scales are scaled to the first axis.

 

Each graph on the diagram has a Y-axis. The diagram has a Y-axis scale, which is separate from the axis. The scale shows the scaling of the axes and is plotted to the left of the first graph. The scale can be longer or shorter than the actual axis. Options for the Y-axes are set in the Y-axes Dialog Box.
