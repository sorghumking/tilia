X-axis Tic Mark Labels

 

Format the Tic Mark Labels on the Tic Marks tab sheet in the X-axes Dialog Box. The following options exist:

  • Distance From Tic Mark: The distance between the bottom of the tic mark and the top of the tic mark label.

  • Divide by: Factor by which to divide the tic mark label. For example, if the label is 1000 and Divide by is 100, the label will be 10. If Divide by is 1000, the label will be 1.

  • Number of Decimal Points: The number of decimal points in the label.

  • Font: Click to change the font of the tic mark labels.

  • Label Origin: If the Label Origin check box is checked, the graph origin will be labeled, otherwise it is not.
